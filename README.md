# Esp32Soundboard

This is my ESP32 Soundboard. 
Small sideproject for the Dokomi 2025. 

## Components used
1. ESP32-S2 DevKit-C1
2. DFPlayer mini
3. Solderd AUX breakout board
4. Soldered Oled White 0,96"
5. 32 GB SD
6. Rotary Encoder with Button Click
7. Rotary Encoder
